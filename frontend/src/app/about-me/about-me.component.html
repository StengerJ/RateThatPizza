Behind the slices

About PGH-Pizza

PGH-Pizza is a Pittsburgh pizza tracking project maintained by Joshua Stenger. It started as a simple way for friends to keep a public record of the places, slices, sauces, crusts, and toppings worth remembering.

What this site is for

The ratings list is meant to be easy to scan and open to everyone. Visitors can browse pizza ratings without logging in, while approved contributors can add new ratings and blog posts as the project grows.

How contributors fit in

People who want to help can apply for a contributor account. Once an admin approves the application, contributors can publish ratings and pizza blog posts directly.

Built for the next step

The frontend is already shaped around the Spring Boot REST API that will eventually store ratings, contributor applications, account data, and blog posts.
